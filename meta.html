Waar ben je allemaal mee bezig?

Ik werk met veel plezier als DevOps Engineer bij Quality Contacts in Zwolle. Daarnaast ben ik altijd bezig met leren. Waar het kan verdiep ik mij in HTML, CSS en Javascript. Ook probeer ik mijn design vaardigheden te verbeteren.

Zo heb ik deze Portfolio website gebouwd. Waar ik eerder veel met libraries als Bootstrap en jQuery werkte. Heb ik er bij deze site voor gekozen om alles zelf te bouwen. Van het lettertype tot het kleurenpalette. Ik heb het allemaal zelf gemaakt!

Op het moment ben ik ook bezig met Figma. Hiermee maak ik de ontwerpen zelf. Eerder hadden mijn websites vooral veel functionaliteit. Nu komt daar ook een strak design bij.
