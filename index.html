Vaardigheid

CSS3

Hetzelfde kan gezegd worden voor CSS3. Met deze iteratie zijn er weer behoorlijk wat "Quality Of Life" improvements gedaan.

Eerder maakte ik veel gebruik van libraries als Bootstrap, Metro of Tailwind CSS. Een tijd geleden heb ik besloten om weer van de grond af CSS te leren. Met onder andere deze site als resultaat.

Waar ben je allemaal mee bezig?

Ik werk met veel plezier als DevOps Engineer bij Quality Contacts in Zwolle. Daarnaast ben ik altijd bezig met leren. Waar het kan verdiep ik mij in HTML, CSS en Javascript. Ook probeer ik mijn design vaardigheden te verbeteren.

Zo heb ik deze website gemaakt. Waar ik eerder veel met libraries als Bootstrap en jQuery werkte. Heb ik er bij deze site voor gekozen om alles zelf te bouwen. Van het lettertype tot het kleurenpalette. Ik heb het allemaal zelf gemaakt!

Op het moment ben ik ook bezig met Figma. Hiermee maak ik de ontwerpen zelf. Eerder hadden mijn websites vooral veel functionaliteit. Nu komt daar ook een strak design bij.
